## Installation

### Build from source

```bash
# Clone the repository
git clone https://github.com/JereIDE/JereIDE.git
cd JereIDE

# or cargo run --release
cargo run

```

> **Note:** This application is optimized for macOS and Windows only. Linux is not planned in the near future.
