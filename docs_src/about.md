# About

JereIDE is a code editor forked from [Lite XL](https://github.com/lite-xl/lite-xl), rewritten from the ground up in Rust with SDL3.

## Origins

Lite XL is a C-based lightweight editor inspired by [lite](https://github.com/rxi/lite). JereIDE replaced the entire codebase with Rust -- rendering, editing, LSP, terminal, and all UI.
